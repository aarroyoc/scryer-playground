# Scryer Playground

Source of the website at https://play.scryer.pl

It's a web application that runs Scryer Prolog using WASM so you can tinker and experiment with the system

Uses: Scryer Prolog v0.9.3

## Testing

Run the unit tests:

```bash
npm test
```

Run tests in watch mode:

```bash
npm run test:watch
```
